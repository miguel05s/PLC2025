## Resumo
O objetivo deste projeto é encontrar uma String Binária que não inclua a substring "011".

## Resultados
Este código verifica se uma sequência de 0s e 1s não contém "011" e retorna True ou False para cada teste: define  ^(?!.*011)[01]*$, a função aceita_sem_011(s) que usa fullmatch para verificar a string inteira.
