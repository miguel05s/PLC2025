---

## Funcionalidades

O conversor reconhece e transforma os seguintes elementos da sintaxe Markdown:

###  Cabeçalhos
- `# Título` → `<h1>Título</h1>`
- `## Subtítulo` → `<h2>Subtítulo</h2>`
- `### Sub-subtítulo` → `<h3>Sub-subtítulo</h3>`

###  Negrito e Itálico
- `**texto**` → `<b>texto</b>`
- `*texto*` → `<i>texto</i>`
